# 🚀 BOSS-BOX - 老板盒子

> 一个U盘，就是一家完全属于您自己的AI公司——无需网络、绝对私密、插上就用。

## 📖 项目简介

BOSS-BOX 是一款专为中小企业主设计的U盘私有AI助手，让您拥有完全本地化、即插即用、数据绝对安全的专属AI工作伙伴。

## ✨ 核心特性

- 🔐 **绝对隐私** - 数据100%本地，永不上云
- 🚀 **即插即用** - 无需配置，插上即用
- 📱 **离线上班** - 无网络也能用AI
- 🧠 **业务专用** - 外贸邮件、合同分析、客户管理
- 🔄 **无线扩展** - 手机平板都能访问
- 🧩 **插件生态** - 持续扩展功能

## 🎯 快速开始

### 硬件要求
- 支持U盘启动的电脑
- 256GB以上U盘或移动硬盘
- 4核CPU / 8GB RAM 以上配置

### 使用方法
1. 下载最新的BOSS-BOX镜像
2. 使用工具写入U盘
3. 从U盘启动电脑
4. 开始您的私有AI办公

## 🛠️ 技术栈

- **系统层**: Ubuntu Core 24.04, LUKS加密
- **AI引擎**: Ollama, Qwen2.5-1.5B
- **应用层**: Tauri + Rust + SvelteKit
- **数据库**: SQLite + SQLCipher
- **向量库**: ChromaDB
- **移动端**: PWA + WebSocket
